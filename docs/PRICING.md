# Pricing

## Positioning

WebSnap API is for developers and agents that need public webpage data transformed into structured JSON quickly, without browser automation overhead.

## Plans

### Free
- Price: **$0**
- Usage: **100 requests/day**
- Intended for evaluation, prototypes, and light usage
- Public docs access

### Pro
- Price: **$19/month**
- Usage: **10,000 requests/month**
- Higher request allowance
- Better fit for production automations and side projects
- API key-based access

### Business
- Price: **$79/month**
- Usage: **100,000 requests/month**
- Commercial/internal-tool usage
- Priority support
- API key-based access

## Why these tiers

The goal is simple packaging:
- free for testing
- pro for solo builders and small teams
- business for repeated operational usage

## Messaging

Primary positioning line:

> Turn any public URL into structured JSON in one API call.

Differentiators to emphasize:
- one-call extraction
- tech-stack detection
- content + metadata in a single response
- built for agents, pipelines, and enrichment workflows

## Example Buyers

- AI agent platforms
- lead-enrichment systems
- internal research pipelines
- SEO tooling
- lightweight site intelligence workflows
